# 网站截图目录

此目录用于存储网站更新历程的截图。

## 截图命名规范

请按照以下格式命名截图文件：

- `day1-initial.png` - 第1天：网站诞生
- `day2-qqzone.png` - 第2天：QQ空间风格
- `day3-autumn.png` - 第3天：秋樱主题
- `day4-timeline.png` - 第4天：时光轴上线

## 添加新截图步骤

1. 截取网站当前状态的截图
2. 调整图片尺寸为 800x600 或 1200x800（保持比例）
3. 将截图保存到此目录，按上述规范命名
4. 在 `app.js` 中的 `updateHistoryData` 数组添加新的历程记录
5. 提交并推送更改

## 截图建议

- 使用浏览器开发者工具的截图功能（Ctrl+Shift+P → "Capture full size screenshot"）
- 或者使用系统截图工具截取浏览器窗口
- 建议使用 PNG 格式以保持清晰度
- 文件大小控制在 500KB 以内
